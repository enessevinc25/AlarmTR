# Hesap Silme Ekran Görüntüleri

Google Play hesap silme URL sayfası için bu klasöre 4 ekran görüntüsü eklemeniz gerekiyor.

## Dosya İsimleri (sırayla)

1. **01-hesabim-ekrani.png** — Hesabım ekranı (ad, e-posta, Hesabı Sil bölümü)
2. **02-silme-onay-dialog.png** — "Hesabı Sil" onay penceresi (VAZGEÇ / EVET, SİL)
3. **03-sifre-dogrulama.png** — Şifre doğrulama penceresi
4. **04-hesap-silindi.png** — "Hesap silindi" onay mesajı

## Nasıl eklenir?

Aldığınız 4 ekran görüntüsünü bu klasöre kopyalayın ve yukarıdaki isimlerle yeniden adlandırın.
